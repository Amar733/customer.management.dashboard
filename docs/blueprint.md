# **App Name**: Shop Manager Dashboard

## Core Features:

- Dashboard Overview: Provides a real-time summary of key performance indicators (KPIs) like total sales, new orders, and inventory levels.
- Product Management: Allows administrators to add, edit, remove, and categorize products, manage stock levels, and upload product images.
- Order Management: Enables viewing, filtering, updating status (e.g., pending, shipped, delivered), and processing customer orders.
- Customer Management: Provides a centralized view of customer profiles, purchase history, and contact information.
- Sales Analytics & Reporting: Displays graphical reports and detailed statistics on sales trends, product performance, and customer behavior.
- Settings & Configuration: Allows configuration of shop settings such as payment gateways, shipping options, and user roles.

## Style Guidelines:

- Primary brand color: A professional and trustworthy deep blue (#2196F3) for navigation, buttons, and accents.
- Background color: A clean, light grey (#F5F5F5) for the main content areas, providing visual separation.
- Secondary accents: Green (#4CAF50) for success indicators and red (#F44336) for warnings or errors.
- Body and headline font: 'Roboto' (sans-serif) for its modern, legible, and versatile nature, ensuring clarity across all data displays.
- Monospace font: 'Source Code Pro' for any code snippets or technical IDs.
- Use a consistent set of clean, filled or outlined icons (e.g., Material Design Icons) for navigation, actions, and data representation.
- Responsive grid-based layout for dashboards and data tables, ensuring usability on various screen sizes.
- Fixed sidebar navigation for primary menu items and a top header for global actions and user profile.
- Subtle fade-in and slide-in animations for loading data, modal transitions, and navigation changes to enhance user experience without distraction.
